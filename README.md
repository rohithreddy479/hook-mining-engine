# Hook Mining Engine 🚀

A system that extracts and ranks high-performing viral hooks from trending content.

## 🔍 Problem

Creators struggle to write engaging hooks and rely on guesswork.

## 💡 Solution

This tool analyzes trending posts, generates viral hooks, and ranks them based on effectiveness.

## ⚙️ Features

* Fetches trending posts based on niche
* Generates viral-style hooks using AI
* Categorizes hooks (curiosity, question, bold)
* Scores hooks based on virality factors
* Clean UI with copy functionality

## 🧠 How it works

1. Input niche
2. Fetch posts
3. Extract hooks using AI
4. Score and rank hooks
5. Display results

## 🛠️ Tech Stack

* FastAPI (Backend)
* Gemini API (AI)
* HTML/CSS/JS (Frontend)

## ▶️ Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
